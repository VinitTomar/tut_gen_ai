## What clicked:

- Python have built-in container data types such as `lists`, `dictionaries`, `sets` & `tuples`.
- `Numpy` provides a high-performance multi-dimensional array object that is the foundation for numerical computation across most ML/AI python libraries.
- A numpy array is a grid of values of the same data type, indexed by a tuple of integers (non-negative for standard indexing, though negative indices are also supported to count from the end).
- We can index numpy array using another array that have indexes.
- Boolean indexing lets us pick some element of array conditionally.
- `Broadcasting` automatically performs operation between two matrices with different size based on some rules described [here](https://numpy.org/doc/stable/user/basics.broadcasting.html)
